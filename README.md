HOW TO BUILD Pascal
=================

Synopsis
========

System requirements
-------------------

* C++ compiler supporting the C++17 standard


Linux, Mac:
-----------

```
cd <path>/<source_package>
g++ pascal.cpp
```

Windows:
--------

1. Open a command prompt.
2. Ensure that the following tools can be found in the path:
 * Supported compiler (Visual Studio 2019 or later)

```
cd <path>\<source_package>
cl pascal.cpp
```
